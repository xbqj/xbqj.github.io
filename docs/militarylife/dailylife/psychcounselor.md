# 做自己的心理导师

**首先，要上好第一堂心理课。**要学习一些心里学的基本知识和调适心理状态的基本常识、原则和方法，学会自觉运用心理学的基本原理和基本知识进行自我心理调适。认清不健康的心理对自身和集体的危害，增强自觉克服心理问题的主动性，树立正确的价值观，磨砺顽强的意志，以饱满的热情投入到军营生活中去。  

**其次，要增强适应环境的能力。**大家要以积极的心态去面对全新的生活环境，培养乐观的精神，提高承受挫折的能力，尽快适应部队的生活，用积极的心态迈出实现理想的第一步。  

**最后，要树立自信心。**大家都刚离开家庭来到军营，有很多的新兵情绪会不稳定，情感很脆弱，大家的心理承受的能力可能不是很强，但是提高自己的自信心才能很好地度过军营生活。
